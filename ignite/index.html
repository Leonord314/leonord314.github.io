How to use:

1. Go to Warcraft Logs and copy your report URL or ID

2. Paste it above and click "Fetch" to load the report

3. Select a fight and click "Analyze"

4. Select which enemy to view Ignite data for

What this shows:

Tracks the shared Ignite debuff across all fire mages. Shows stack value over time, who owns the Ignite, when it rolls vs drops, per-mage contribution stats, and threat attribution.

Private logs cannot be imported (WCL API restriction).
